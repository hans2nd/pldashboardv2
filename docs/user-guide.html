Memulai

Login ke Aplikasi
1
Buka Halaman Login

Akses URL aplikasi yang diberikan oleh administrator. Anda akan melihat halaman login.

2
Masukkan Email & Password

Gunakan email dan password yang telah diberikan oleh administrator.

3
Klik Tombol Login

Setelah berhasil login, Anda akan diarahkan ke halaman dashboard utama.

Tips: Jika lupa password, hubungi administrator untuk reset password.

Navigasi Aplikasi

Sidebar Menu

Menu utama aplikasi berada di sebelah kiri. Klik menu untuk membuka dashboard atau fitur.

Submenu

Beberapa menu memiliki submenu. Klik menu induk untuk melihat daftar submenu.

Profile Dropdown

Klik nama Anda di pojok kanan atas untuk akses profil dan logout.

Breadcrumb

Navigasi breadcrumb di atas halaman menunjukkan posisi Anda saat ini.

Info: Menu yang Anda lihat tergantung pada hak akses (permission) yang diberikan oleh administrator.

Melihat Dashboard

Dashboard menampilkan data dalam bentuk visual (grafik, tabel, dll) untuk memudahkan analisis.

Cara Melihat Dashboard
1
Pilih Menu Dashboard

Klik menu dashboard yang ingin Anda lihat dari sidebar (contoh: Sales Dashboard → Sidoarjo → Distributor)

2
Tunggu Loading

Dashboard akan dimuat dalam iframe. Tunggu beberapa saat hingga data tampil sepenuhnya.

3
Interaksi dengan Dashboard

Anda dapat berinteraksi dengan dashboard: filter data, klik grafik untuk detail, dll (tergantung konfigurasi).

Fitur Dashboard
Fitur Keterangan
Filter Beberapa dashboard memiliki filter untuk menyaring data berdasarkan periode, kategori, dll
Fullscreen Klik tombol fullscreen (jika ada) untuk melihat dashboard dalam layar penuh
Refresh Klik tombol refresh atau tekan F5 untuk memuat ulang data terbaru
Export Beberapa dashboard mendukung export data ke PDF atau Excel (tergantung konfigurasi)
Tips: Jika dashboard tidak tampil atau error, coba refresh halaman atau hubungi administrator.

Profil & Password

Mengubah Password
1
Klik Nama Anda

Di pojok kanan atas, klik nama atau foto profil Anda untuk membuka dropdown menu.

2
Pilih "Ubah Password"

Klik opsi "Ubah Password" atau "Change Password" dari menu dropdown.

3
Isi Form

Masukkan password lama, password baru, dan konfirmasi password baru.

4
Simpan

Klik tombol "Simpan" untuk menyimpan password baru Anda.

Perhatian:
  • Password minimal 8 karakter
  • Password baru tidak boleh sama dengan password lama
  • Simpan password Anda dengan aman
Logout

Untuk keluar dari aplikasi:

  1. Klik nama Anda di pojok kanan atas
  2. Pilih "Logout" dari menu dropdown
  3. Anda akan diarahkan ke halaman login
Tips Keamanan: Selalu logout setelah selesai menggunakan aplikasi, terutama di komputer bersama.

FAQ (Pertanyaan Umum)

Bagaimana jika saya lupa password?
Hubungi administrator untuk mereset password Anda. Administrator dapat mengubah password dari menu Users.
Mengapa saya tidak bisa melihat menu tertentu?
Menu yang tampil tergantung pada hak akses (permission) Anda. Jika Anda memerlukan akses ke menu tertentu, hubungi administrator untuk meminta penambahan permission.
Dashboard tidak tampil atau error, apa yang harus dilakukan?
  1. Coba refresh halaman dengan menekan F5
  2. Clear cache browser
  3. Coba buka di browser lain
  4. Jika masih error, hubungi administrator
Data dashboard tidak update, bagaimana cara refresh?
Tekan F5 atau klik tombol refresh di browser. Data dashboard biasanya diupdate secara otomatis sesuai jadwal yang dikonfigurasi oleh administrator.
Apakah saya bisa mengakses dari smartphone?
Ya, aplikasi ini responsif dan dapat diakses melalui browser di smartphone atau tablet. Namun, untuk pengalaman terbaik, disarankan menggunakan layar yang lebih besar.
